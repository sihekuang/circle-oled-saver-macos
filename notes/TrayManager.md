# TrayManager

**File**: `CircleApp/TrayManager.swift`

Manages the `NSStatusItem` (menu bar icon) and its context menu.

## Menu Items

1. **Status toggle** - shows enabled/disabled state (green/red circle)
2. **Always On** - toggle with checkmark indicator
3. **Settings...** - opens `SettingsWindowController`
4. **Quit Circle** - terminates the app

## Icon

Uses `TrayIcon` template image from `Assets.xcassets`.
